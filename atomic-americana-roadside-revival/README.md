# Atomic Americana: Roadside Revival

A mobile-friendly HTML5 match-3 prototype set in the Atomic Americana universe.

## Current playable features

- 8x8 match-3 board
- Tap-to-swap and swipe controls
- Cascades and scoring
- 25-move level
- Hint system
- Automatic reshuffle when no move remains
- Local high-score saving
- Responsive phone layout

## Current level

Restore the Moonbeam Diner by scoring 2,500 points.

## Run locally

Open `index.html` in a browser, or serve this folder through GitHub Pages.
